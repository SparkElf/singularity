## 現在の状況

Singularity は現在も活発に開発されており、まだ本番運用向けの正式リリースではありません。設計文書や計画文書に記載された機能は、未完成または利用できない場合があります。実際の状況は、現在のソースコードと検証結果を基準としてください。

## プロジェクトの方向性

本プロジェクトは SiYuan を基盤として、組織、共有、権限、コラボレーション機能を備えたクラウド型エンタープライズナレッジベースへの発展を目指しています。これらは開発目標であり、すべての機能が実装済みであることを示すものではありません。

現在は、保守可能な独立フォーク、明確な上流取り込み手順、一貫したデザインシステム、将来のエンタープライズ機能に必要なアイデンティティとスペースの基盤を優先しています。
